Buttons

Use Windy's custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Examples

Windy includes several predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control.

Buttons

+button(kind="primary") Primary Button
+button(kind="secondary") Secondary Button
+button(kind="ghost") Ghost Button

Large Buttons

+button(kind="primary" size="large") Large Primary Button
+button(kind="secondary" size="large") Large Secondary Button
+button(kind="ghost" size="large") Large Ghost Button

Disabled Buttons

+button(kind="primary" disabled) Large Primary Button
+button(kind="secondary" disabled) Large Secondary Button
+button(kind="ghost" disabled) Large Ghost Button

Dialog

Use dialogs...

Show

+dialog#dialog1
  +button(kind="primary" data-action="close" size="large") Close Dialog
+button(kind="primary" data-target="#dialog1" size="large" data-action="show") Show Dialog

Show Modal

+dialog#dialog2
  +button(kind="primary" data-action="close" size="large") Close Dialog
+button(kind="primary" data-target="#dialog2" size="large" data-action="showModal") Show Modal Dialog

Forms

Examples and usage guidelines for form control styles, layout options, and custom components for creating a wide variety of forms.

Examples

Give textual form controls like inputs and textareas an upgrade with custom styles, sizing, focus states, and more..

Input Fields

+textInputField(name="text1" placeholder="Text")
+emailInputField(name="email1" placeholder="Email")

Input Fields with Labels

+textInputField(label="Text 1" name="text1" id="text1" placeholder="Text")
+emailInputField(label="Email 1" name="email1" id="email1" placeholder="Email")

Select Field

+selectField(name="city" id="city")
  +option(value="berlin") Berlin
  +option(value="copenhagen") Copenhagen
  +option(value="london") London

Select Field with Labels

+selectField(label="City" name="city" id="city")
  +option(value="berlin") Berlin
  +option(value="copenhagen") Copenhagen
  +option(value="london") London

Form
